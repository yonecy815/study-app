# データベースセットアップ手順

## 📋 このセットアップで有効になる機能

以下の機能を使用するには、初回のみデータベースセットアップが必要です。

- ✅ 招待コードによる生徒登録
- ✅ 先生ごとの生徒数制限管理
- ✅ 先生モードの暗証番号設定
- ✅ 宿題の配信・管理機能

## 📋 セットアップ手順

### ステップ1: Supabaseダッシュボードにアクセス

1. ブラウザで [https://supabase.com/dashboard](https://supabase.com/dashboard) を開く
2. プロジェクトを選択
3. 左サイドバーの **SQL Editor** をクリック

### ステップ2: SQLスクリプトを実行

1. **New Query** ボタンをクリック
2. `setup-database.sql` ファイルの内容を**すべてコピー**
3. SQL Editorに**ペースト**
4. 右上の **Run** ボタンをクリック

### ステップ3: 実行結果の確認

成功すると以下のメッセージが表示されます：
```
Database setup completed successfully!
```

### ステップ4: テーブル構造の確認

左サイドバーの **Table Editor** で以下を確認：

#### profiles テーブル
以下のカラムが追加されているか確認：
- ✅ `teacher_id` (uuid)
- ✅ `invite_code` (text)
- ✅ `is_teacher` (boolean)
- ✅ `teacher_password` (text)
- ✅ `student_limit` (integer)
- ✅ `has_unlimited_license` (boolean)

#### homeworks テーブル
新しく作成されたテーブル：
- ✅ `id` (bigserial)
- ✅ `student_id` (uuid)
- ✅ `folder_name` (text)
- ✅ `assigned_date` (date)
- ✅ `created_at` (timestamp)

## 🎯 セットアップ後の動作確認

### 1. 先生アカウントの作成

1. アプリにアクセス
2. 新規登録（通常の方法で）
3. ログイン後、⚙️ アイコンをクリック
4. デフォルトパスワード `testpass` を入力して先生モードを解除
5. 招待コードが自動生成されます

### 2. 生徒の登録

1. 別のブラウザまたはシークレットモードで開く
2. 新規登録時に**先生の招待コード**を入力
3. 登録完了

### 3. 動作確認項目

- ✅ 先生モードで生徒一覧が表示される
- ✅ 招待コードが表示される
- ✅ 生徒数が「X / 3人」と表示される
- ✅ 宿題を設定できる
- ✅ 生徒に問題をアップロードできる

## 🔐 セキュリティ設定

このSQLスクリプトは以下のセキュリティポリシーを設定します：

### Row Level Security (RLS)

1. **profiles テーブル**
   - ユーザーは自分のプロフィールを閲覧・更新可能
   - 先生は自分の生徒のプロフィールを閲覧可能
   - 誰でも招待コードで先生を検索可能（登録時のみ）

2. **homeworks テーブル**
   - 生徒は自分の宿題のみ閲覧可能
   - 先生は自分の生徒の宿題を管理可能

3. **problems テーブル**
   - ユーザーは自分の問題を管理可能
   - 先生は自分の生徒の問題を管理可能

## ⚠️ トラブルシューティング

### エラー: "relation already exists"
→ すでにテーブルやカラムが存在する場合は無視してOK（`IF NOT EXISTS`で保護されています）

### エラー: "permission denied"
→ Supabaseダッシュボードで実行する必要があります（ローカルからは実行できません）

### 先生モードのパスワードが動作しない
→ デフォルトパスワード: `testpass`

### 無制限ライセンスを有効化したい
→ 先生モードに入った後、マイページで無制限ライセンスの暗証番号入力欄に `testpass` を入力

## 📝 追加情報

### デフォルト設定

- 生徒数制限: **3人**
- 先生モードのデフォルトパスワード: `testpass`
- 無制限ライセンスのパスワード: `testpass`

### カスタマイズ

先生ごとに異なる設定を行う場合は、Table Editorで直接編集できます：

1. **Table Editor** → **profiles** テーブル
2. 対象の先生ユーザーを選択
3. 以下の値を編集：
   - `student_limit`: 生徒数上限を変更
   - `teacher_password`: 先生モード用の独自パスワードを設定
   - `has_unlimited_license`: true に設定すると無制限
